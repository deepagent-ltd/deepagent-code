# Contributing

Thank you for contributing to DeepAgent Code.

## Development

```bash
bun install
bun run typecheck
bun run --cwd packages/deepagent-code test
```

Run narrower package tests when possible and include the commands you ran in your pull request.

## DCO

By contributing, you certify the Developer Certificate of Origin 1.1 below. Sign off each commit with:

```bash
git commit -s
```
