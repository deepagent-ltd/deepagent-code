## PodeŇ°avanje ureńĎivańća

Obje naredbe `/editor` i `/export` koriste editor specificiran u vaŇ°oj varijabli okruŇĺenja `EDITOR`.

<Tabs>
  <TabItem label="Linux/macOS">

    ```bash

    export EDITOR=nano
    export EDITOR=vim

    # For GUI editors, VS Code, Cursor, VSCodium, Windsurf, Zed, etc.
    # include --wait
    export EDITOR="code --wait"
    ```

    Da biste ga uńćinili trajnim, dodajte ovo u svoj shell profil;
    `~/.bashrc`, `~/.zshrc`, itd.

  </TabItem>

  <TabItem label="Windows (CMD)">

    ```bash
    # For GUI editors, VS Code, Cursor, VSCodium, Windsurf, Zed, etc.
    # include --wait
    set EDITOR=code --wait
    ```

    Da biste ga uńćinili trajnim, koristite **Svojstva sistema** > **OkruŇĺenje
    Varijable**.

  </TabItem>

  <TabItem label="Windows (PowerShell)">

    ```powershell

    # For GUI editors, VS Code, Cursor, VSCodium, Windsurf, Zed, etc.
    # include --wait
    $env:EDITOR = "code --wait"
    ```

    Da biste ga uńćinili trajnim, dodajte ovo u svoj PowerShell profil.

  </TabItem>
</Tabs>

## Konfiguracija

MoŇĺete prilagoditi TUI ponaŇ°anje putem `tui.json` (ili `tui.jsonc`).

```json title="tui.json"
  "leader_timeout": 2000,
  "keybinds": {
```

`keybinds` se spaja s ugrańĎenim zadanim vrijednostima, tako da trebate konfigurisati samo preńćice koje Ňĺelite promijeniti.

### Opcije

- `theme` - Postavlja vaŇ°u UI temu. [Saznajte viŇ°e](/docs/themes).
- `keybinds` - PrilagońĎava preńćice na tastaturi. [Saznajte viŇ°e](/docs/keybinds).
- `leader_timeout` - KontroliŇ°e koliko dugo DeepAgent Code ńćeka nakon leader key. Podrazumijevano je `2000`.
- `scroll_acceleration.enabled` - ‚Äč‚ÄčOmoguńáite ubrzanje pomicanja u macOS stilu za glatko, prirodno pomicanje. - `scroll_speed` - ‚Äč‚ÄčKontrolira koliko brzo TUI skroluje kada se koriste komande za pomeranje (minimum: `0.001`, podrŇĺava decimalne vrijednosti). Podrazumevano je `3`. **Napomena: Ovo se zanemaruje ako je `scroll_acceleration.enabled` postavljeno na `true`.**
- `diff_style` - Kontrolira prikazivanje razlike. `"auto"` se prilagońĎava Ň°irini terminala, `"stacked"` uvijek prikazuje raspored u jednoj koloni.
- `mouse` - Omoguńáava ili onemoguńáava hvatanje miŇ°a u TUI (podrazumijevano: `true`). Kada je onemoguńáeno, zadrŇĺava se izvorno ponaŇ°anje terminala za oznańćavanje i pomicanje miŇ°em.
- `attention` - KonfiguriŇ°e desktop obavjeŇ°tenja i zvukove za TUI. Podrazumijevano je onemoguńáeno.

Koristite `DEEPAGENT_CODE_TUI_CONFIG` da uńćitate prilagońĎenu putanju TUI konfiguracije.

### Attention

Attention omoguńáava da vas TUI obavijesti kada DeepAgent Code ńćeka odgovor, treba odobrenje dozvole, prijavi greŇ°ku sesije ili zavrŇ°i sesiju. Omoguńáite ga pomońáu `attention.enabled`; ugrańĎeni dogańĎaji reproduciraju zvuk kada se dogode. Desktop obavjeŇ°tenja se Ň°alju samo kada prozor terminala nije u fokusu i ne koriste se za subagent dogańĎaje.

- `enabled` - Omoguńáava sva obavjeŇ°tenja i zvukove za Attention. Podrazumijevano je `false`.
- `notifications` - Kada je Attention omoguńáen, dozvoljava TUI-ju da Ň°alje desktop obavjeŇ°tenja putem terminala. Podrazumijevano je `true`.
- `sound` - Kada je Attention omoguńáen, dozvoljava zvukove upozorenja. Podrazumijevano je `true`.
- `volume` - Podrazumijevana jańćina zvukova upozorenja od `0` do `1`. Podrazumijevano je `0.4`.
- `sound_pack` - Sound pack ID koji se koristi. Podrazumijevano je `deepagent-code.default`.
- `sounds` - Postavlja prilagońĎene zvuńćne datoteke za `default`, `question`, `permission`, `error`, `done` ili `subagent_done`. Putanje mogu biti apsolutne, `file://` URL-ovi ili relativne u odnosu na `tui.json`.
