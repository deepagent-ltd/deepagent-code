# React Frontend Engineering

## Boundary

React component behavior, hooks, state, effects, rendering, hydration, routing, and React test patterns.

## Out of Scope

Generic DOM/CSS/a11y belongs to frontend.web/accessibility; API contracts belong to backend.api.

## Evidence Rules

All documents are medium-evidence system seeds. Professional, regulated, or high-risk decisions require human authority. Failure dossiers are not indexed.

## L3 Validation

Representative activation and retrieval smoke lives in `evals/smoke/l3-smoke.json`. The quality report lives in `quality/l3-report.json`. A pack reaches L3 only when activation, pack-scoped retrieval, validation smoke, diagnosis smoke, and failure dossier boundaries are all present.
